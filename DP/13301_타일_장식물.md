# 타일 장식물
[문제](https://www.acmicpc.net/problem/13301)

### 문제 이해
- 주어진 조건 
  - 타일의 개수 N(1 ≤ N ≤ 80)  
  - 타일 한변의 둘레 : 1, 1, 2, 3, 5, 8...   
- 필요한 결과 : N개의 타일로 구성된 직사각형의 둘레

### 재정의, 추상화
타일 1개 -> 4  
타일 2개 -> 6  
타일 3개 -> 10  
타일 4개 -> 16    
타일 5개 -> 26  
타일 6개 -> 42  

### 사용할 알고리즘과 자료구조 선택
둘레 규칙은 피보나치 수열
fn = (fn - 1) + (fn - 2)  

### 계획 검증(걸리는 시간, 메모리 확인)
N의 범위가 80이라서 int로는 불가능, long 사용
