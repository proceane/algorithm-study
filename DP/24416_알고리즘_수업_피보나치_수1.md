# 알고리즘 수업 - 피보나치 수 1
[문제](https://www.acmicpc.net/problem/24416)

### 문제 이해
- 주어진 조건  
  - N(5 ≤ n ≤ 40)  
- 필요한 결과 : n의 피보나치 수를 재귀호출과 동적프로그래밍으로 실행했을 때 각 코드의 실행 횟수
