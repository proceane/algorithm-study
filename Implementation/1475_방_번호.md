# 방 번호
[문제](https://www.acmicpc.net/problem/1475)

### 문제 이해
- 주어진 조건   
  - 첫째 줄에 방 번호 N이 주어짐  
    - N은 1,000,000보다 작은 자연수
- 필요한 결과 : 필요한 세트의 개수 출력

### 재정의, 추상화
1. 필요한 숫자의 개수를 파악  
2. 개수를 충족시키기 위한 숫자 세트의 개수 카운트  

한 세트 당 0-9까지 있음  
6이랑 9는 뒤집어서 쓸 수 있음  

0부터 9까지 순회  
1차로 하나씩 뺌  
숫자가 남아있으면 한번 더 순회  
숫자가 안남을때까지 계속 순회  
순회한 개수 = 카드 세트 개수

### 사용할 알고리즘과 자료구조 선택
특정한 알고리즘이 필요한건 아닌듯  
자료구조는 배열  

### 계획 검증(걸리는 시간, 메모리 확인)
N의 최대 숫자가 백만이기도 하고 시간제한도 2초라서 괜찮을지도..?
