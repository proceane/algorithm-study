# 다음수
[문제](https://www.acmicpc.net/problem/4880)

### 문제 이해
- 주어진 조건  
  - 각 테스트케이스마다 수열의 연속하는 세 정수 a1, a2, a3이 한번에 주어짐  
  - 입력의 마지막 줄에는 0이 세개 주어짐  
- 필요한 결과 : 등차수열이면 AP를, 등비수열이면 GP를 출력한뒤 다음 항을 출력  

### 재정의, 추상화
a2 - (a3 - a2) = a1이면 등차수열  
a2 / (a3 / a2) = a1이면 등비수열   

### 사용할 알고리즘과 자료구조 선택
특정한 알고리즘이나 자료구조는 사용하지 않을 듯  

### 계획 검증(걸리는 시간, 메모리 확인)
숫자가 3개만 주어지기때문에 시간초과는 안생길듯
